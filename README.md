## 📝 To-Do List App (React Native - AI Vibe Coding)

This project is a To-Do application developed using an **AI-assisted vibe coding approach**. A comprehensive and easy-to-use mobile application built with **React Native** and **Expo**. This app allows users to create, manage, and track daily, monthly, and yearly plans and tasks.

The goal of this project is to experiment with rapid prototyping and mobile development using AI tools.

### ✨ Features
- **Categorized Tasks:** Manage your plans easily with Daily, Monthly, and Yearly tabs.
- **Task Creation:** Add new tasks with specific date selection.
- **Progress Tracking:** Track the completion status of your tasks easily.
- **Simple mobile-first UI:** Designed for a smooth mobile experience.
- **Data Persistence:** All your tasks are securely saved locally on your device using `AsyncStorage`.
- **Modern Navigation:** Built with Expo Router for seamless, file-based routing and a great user experience.

### 🛠️ Requirements & Technologies
To run the project you need:
- Node.js
- npm or yarn
- Expo
- Expo Go mobile app

- **Framework:** [React Native](https://reactnative.dev/)
- **Toolset:** [Expo](https://expo.dev/)
- **Navigation:** [Expo Router](https://docs.expo.dev/router/introduction/)
- **Storage:** [@react-native-async-storage/async-storage](https://react-native-async-storage.github.io/async-storage/)

### 🚀 Installation & Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/birgoraliahmet/to-do-list.git
   cd to-do-list
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the project:**
   ```bash
   npx expo start
   ```

4. **Run on your device:**
   - Scan the QR code with the **Expo Go** app on your mobile device to run the application.
   - Alternatively, press `a` for Android emulator or `i` for iOS simulator.

## 📝 Yapılacaklar Listesi Uygulaması (React Native - AI Vibe Coding)

Bu proje, **AI destekli vibe coding yaklaşımı** kullanılarak geliştirilmiş bir To-Do uygulamasıdır. **React Native** ve **Expo** kullanılarak geliştirilmiş, kapsamlı ve kullanımı kolay bir mobil Yapılacaklar Listesi uygulaması. Bu uygulama kullanıcıların günlük, aylık ve yıllık planlarını oluşturmasına, yönetmesine ve takip etmesine olanak tanır.

Projenin amacı, yapay zeka ile hızlı prototipleme ve mobil uygulama geliştirme sürecini denemektir.

### ✨ Özellikler
- **Kategorize Edilmiş Görevler:** Günlük, Aylık ve Yıllık sekmeler ile planlarınızı kolayca yönetin.
- **Görev Oluşturma:** Belirli tarih seçimleriyle detaylı yeni görevler ekleyin.
- **İlerleme Takibi:** Görevlerinizin tamamlanma durumunu tek dokunuşla işaretleyin ve takip edin.
- **Basit ve mobil odaklı arayüz:** Akıcı bir mobil deneyim için tasarlandı.
- **Veri Kalıcılığı:** `AsyncStorage` kullanılarak tüm görevleriniz çevrimdışı olarak cihazınıza güvenle kaydedilir.
- **Modern Arayüz:** Expo Router ile akıcı, dosya tabanlı ve kesintisiz sayfa geçişleri.

### 🛠️ Gereksinimler & Teknolojiler
Projeyi çalıştırmak için:
- Node.js

- Expo
- Expo Go mobil uygulaması
